### Photonvision
Photonvision is a new vision system that we switched to for the 2026 season, we utilized 4 OV-9281 Arducam Cameras, and 2 Orange pi 5 16gb's, using it to automatically align with the hub and the tower, as well as shoot on the fly.

### On-The-Fly Shooting
This year we will have access to on the fly shooting, where we will be able to move and shoot at the same time, while the robot automatically compensates for directional momentum on the ball so it goes in no matter what.
